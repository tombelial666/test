# Clearing Systemactions Int2 examples

## Default run

```bash
(cwd: qa/Tools/ClearingTester) python qa/Tools/ClearingTester/run_volant_easy_to_borrow_int2.py
```

## With explicit environment

```bash
export CLEARING_TESTER_USERNAME=<value>
export CLEARING_TESTER_PASSWORD=<value>
export CLEARING_TESTER_APP_KEY=<value>
(cwd: qa/Tools/ClearingTester) python qa/Tools/ClearingTester/run_volant_easy_to_borrow_int2.py
```

## With explicit environment (PowerShell)

```powershell
$env:CLEARING_TESTER_USERNAME="<value>"
$env:CLEARING_TESTER_PASSWORD="<value>"
$env:CLEARING_TESTER_APP_KEY="<value>"
(cwd: qa/Tools/ClearingTester) python qa/Tools/ClearingTester/run_volant_easy_to_borrow_int2.py
```
