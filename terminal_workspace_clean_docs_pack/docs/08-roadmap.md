# 08 — Roadmap

## Phase 1 — Clean stability

- stable ASCII-safe UI
- working profiles
- working profile-aware cheats
- working direct hotkeys
- healthcheck script

## Phase 2 — Hacker theme

Add optional themes:

```text
clean
toxic-hacker
low-gpu
```

Requirements:
- no mandatory Nerd Font glyph dependency;
- switchable by one variable or one config line;
- no profile behavior changes;
- no cheatsheet logic changes;
- low-gpu mode disables Acrylic/opacity/gradient.
