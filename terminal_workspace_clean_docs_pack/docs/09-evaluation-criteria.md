# 09 — Критерии оценки (13×13)

Шкала: **0..13** для каждого пункта.

Ориентир:
- **0** — отсутствует / ломает базовый сценарий
- **7** — приемлемо, но есть ощутимые проблемы/долги
- **10** — хорошо, соответствует лучшим практикам для “рабочего daily”
- **13** — образцово: безопасно, устойчиво, удобно, расширяемо

## 1) Стабильность (crash-free / regression-free)
Оцени: падения WezTerm/профилей, “рандомные” поломки, регрессии после изменений.

## 2) Предсказуемость поведения
Оцени: одинаковые действия дают одинаковый результат; нет “магии” и неожиданных side-effect.

## 3) Разделение Windows vs WSL
Оцени: документация/cheats не смешивают команды; переходы между средами понятны.

## 4) Профили и `AI_TERM_PROFILE`
Оцени: все профили есть, переменная выставляется корректно, проверка проста.

## 5) Cheatsheet-интеграция (profile-aware)
Оцени: `cheat`/hotkey показывает **правильный** файл под профиль, без путаницы.

## 6) Безопасность (опасные команды и defaults)
Оцени: наличие “опасных” примеров (`rm -rf`, `Remove-Item -Recurse -Force`) и то, как они оформлены/ограждены.

## 7) Non-exec по умолчанию
Оцени: Navi работает в режиме `--print`; нет авто-исполнения из cheats.

## 8) UX хоткеев (нет дублей, логичная карта)
Оцени: нет 3 хоткеев на одно действие; есть отдельная “кнопка” на cheatsheet; конфликты минимальны.

## 9) Читаемость UI (ASCII-first)
Оцени: табы/статус/промпт читаемы без Nerd Font; корректные кнопки окна.

## 10) Производительность (low-gpu friendly)
Оцени: отсутствие лишней нагрузки; возможность отключить Acrylic/opacity/анимации.

## 11) Установка и миграция (идемпотентность)
Оцени: install-скрипт повторяемый, не ломает систему, создаёт бэкапы, корректно работает “поверх” существующих файлов.

## 12) Диагностика (healthcheck и наблюдаемость)
Оцени: здоровье окружения проверяется быстро; понятно что сломано и как чинить.

## 13) Документация и воспроизводимость
Оцени: новичок может поднять окружение по докам; экспорт/импорт понятен; нет “скрытых шагов”.
