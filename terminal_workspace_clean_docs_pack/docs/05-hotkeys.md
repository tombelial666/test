# 05 — Хоткеи

## Launcher

| Action | Hotkey |
|---|---|
| Open profile launcher | F2 |
| Fallback launcher | Alt+L |
| Command palette | Ctrl+Shift+P |
| Quick select | Ctrl+Shift+Space |
| Cheatsheet (profile-aware) | Ctrl+Shift+/ |

## Прямые профили

| Profile | Hotkey |
|---|---|
| PowerShell | Ctrl+Alt+1 |
| CMD | Ctrl+Alt+2 |
| Ubuntu Bash | Ctrl+Alt+3 |
| Ubuntu Zsh | Ctrl+Alt+4 |
| Ubuntu Fish | Ctrl+Alt+5 |

## Панели (panes)

Leader:

```text
Ctrl+A
```

| Action | Hotkey |
|---|---|
| Split vertical | Ctrl+A, then s |
| Split horizontal | Ctrl+A, then d |
| Close pane | Ctrl+A, then x |
| Zoom pane | Ctrl+A, then z |
| Move left/down/up/right | Ctrl+A, then h/j/k/l |
| Resize | Ctrl+A, then arrows |

## Вкладки (tabs)

| Action | Hotkey |
|---|---|
| Previous tab | Alt+Left |
| Next tab | Alt+Right |
| New tab | Ctrl+Shift+N |
| Close tab | Ctrl+Shift+W |
