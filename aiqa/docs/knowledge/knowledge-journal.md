# Knowledge Journal — промоции находок в канон

> Сюда попадают находки из `tasks/*/discoveries.md` которые промотированы в canonical слой.
> Это лог решений: что добавили, откуда пришло, почему.
> Новые записи добавляются сверху.

---

## Как пользоваться

**Добавить запись** (после промоции YAML в repo-index или impact-map):

```
1. Скопируй блок "Запись" ниже
2. Заполни поля
3. Добавь в начало списка (под горизонтальной линией)
4. Обнови discoveries.md: статус draft → promoted
5. Закоммить вместе с изменением в canonical YAML
```

**Найти откуда пришло правило:**
Каждое правило в `impact-map.yaml` имеет `evidence_basis: task_discovery` → ищи здесь по task-id.
