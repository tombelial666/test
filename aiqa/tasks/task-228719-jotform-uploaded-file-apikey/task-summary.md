# task-summary — 228719

## Суть простым языком

JotForm перестал отдавать файлы по прямым ссылкам `…/uploads/…` без авторизации: в ответ приходит **HTML страница логина**. AMS скачивал файл как байты и передавал дальше в обработку изображений (Snap и т.д.), из‑за чего заявки ломались или «застревали».

## Бизнес-намерение

Восстановить успешное открытие счёта для сценариев с загрузками из JotForm (в т.ч. Sogo, foreign, US citizen), не меняя продуктовую модель форм.

## Техническое намерение

При `GET` на URL загруженного файла передавать **валидный JotForm API key** так, как требует текущий API (в известной реализации — заголовок `APIKEY` в `DownloadUploadedFileAsync`).

## Поведение

- **Добавлено/изменено:** исходящий запрос на скачивание файла включает аутентификацию JotForm.
- **Защищается:** от повторной выдачи HTML вместо бинарника/картинки на этом шаге.
- **Не заявлено в scope:** исправление 502/HTML от **Apex Snap** — отдельная линия расследования.
