## TC-228719-01 — E2E: одно вложение JotForm

- **Предусловия:** Стенд с фикс-сборкой AMS; форма с одним file upload; ключ JotForm валиден.
- **Шаги:** Пройти отправку заявки до обработки clearing/onboarding.
- **Ожидание:** Нет HTML login от JotForm; изображение уходит в Snap без ошибки парсинга на шаге скачивания.
- **Проверка:** Логи AMS + успешный статус заявки (или ожидаемая ошибка только на следующем шаге, не JotForm HTML).
